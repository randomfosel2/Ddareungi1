# Ddareungi1
따릉이 연습1
🚲 서울시 따릉이 대여소 인프라와 운영 효율 분석
📌 프로젝트 개요
서울시 공공자전거 '따릉이' 데이터를 활용하여 자치구별 대여소 운영 현황을 파악하고, 대여소 인프라의 규모가 운영 효율에 미치는 영향을 분석한 프로젝트입니다.

설정 가설: "대여소가 많은 자치구일수록 대여소 1개당 대여건수(효율)가 높을 것이다." (네트워크 효과 존재 여부 확인)

📊 데이터셋 설명
데이터 파일: 따릉이_분석용.csv

데이터 크기: 13,795행 × 8열

수집 기간: 2025년 7월 ~ 2025년 11월

주요 컬럼: 자치구, 대여소명, 기준년월, 대여건수, 반납건수, 위도, 경도, 거치대수

🔍 주요 분석 및 결과
1. 자치구별 대여건수 및 대여소 현황
서울시 총 25개 자치구를 대상으로 분석을 진행했습니다.

총 대여건수 1위: 강서구 (2,200,391건)

대여건수 최다 대여소: 강서구 '2715.마곡나루역 2번 출구' (개별 기준 최고 17,664건 기록)

대여소 개수 1위: 송파구 (221개)

2. 대여소 운영 효율 분석
각 자치구의 대여건수 합계를 대여소 개수로 나누어 **대여소 1개당 평균 대여건수(대여 효율)**를 산출했습니다.

대여 효율 TOP 3 자치구

강서구: 약 11,169건 (대여소 197개)

양천구: 약 9,872건 (대여소 111개)

영등포구: 약 9,426건 (대여소 173개)

반면, 대여소 개수가 가장 많은 송파구는 대여 효율 부문에서 5위(약 7,442건)를 기록했습니다.

3. 가설 검증 (상관관계 분석)
방법: 자치구별 '대여소 개수'와 '대여소 1개당 대여건수(효율)' 간의 피어슨 상관계수(Pearson Correlation)를 계산하고 회귀선으로 시각화했습니다.

결과: 상관계수(r)가 약 0.49로 산출되어 양의 상관관계가 존재함을 확인했습니다.

결론: 대여소 인프라가 클수록 효율도 함께 증가하는 '네트워크 효과'가 어느 정도 존재한다고 볼 수 있으나, 완전히 뚜렷한 강한 상관관계는 아니므로 타 요인(입지, 유동인구 등)도 영향을 미치는 것으로 분석됩니다.

🛠 사용 기술 (Tech Stack)
Language: Python 3

Libraries: Pandas (데이터 전처리), Matplotlib & Seaborn (시각화), SciPy (통계 및 상관관계 분석)

Environment: Google Colab
